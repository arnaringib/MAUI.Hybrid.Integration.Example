# MAUI.Hybrid.Integration.Example
This is an example of how to perform integration/e2e tests on MAUI Blazor Hybrid application with Appium and Playwright. Please note that this example only testing Windows application.

# Prerequisite 
1. Setup Appium (https://appium.io/docs/en/latest/quickstart/install/) (I used version 1.2.1)
2. Setup Windows Application driver (https://github.com/microsoft/WinAppDriver)
3. Setup Playwright (https://playwright.dev/docs/intro)
